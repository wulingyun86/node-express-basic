# node-express-basic
这个项目采用nodejs + express + mongoose + swig + jquery + bootstrap 搭建的，前后台不分离的 简单的 交互式框架
操作步骤:
1， git clone https ~~ 将项目下载到本地
2， cd node-express-basic  进入项目
3,  npm install 初始化
4， mongod --dbpath=dbxxxxxxxx   配置mongodb
5,  node main 启动
6，localhost:8080 直接访问
